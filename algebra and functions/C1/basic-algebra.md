#### 9. Linear equations with brackets

```
3(2x - 1) = 4(x - 2)  
∴ 6x - 3 = 4x - 8  
∴ 6x = 4x - 5  
∴ 2x = -5  
∴ x = -2.5
```

```
7 - 5(3x - 4) = 3 - (2x + 5)**  
∴ 7 - 15x + 20 = 3 - 2x - 5  
∴ 27 - 15x = -2 - 2x  
∴ 27 = -2 + 13x  
∴ 29 = 13x  
∴ 29/13 = x  
∴ 2 3/13 = x  
∴ x = 2 3/13  
```
