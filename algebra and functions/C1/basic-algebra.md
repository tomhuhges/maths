### 9. Linear equations with brackets

**3(2x - 1) = 4(x - 2)**

```
∴ 6x - 3 = 4x - 8  
∴ 6x = 4x - 5  
∴ 2x = -5

∴ x = -2.5
```

**7 - 5(3x - 4) = 3 - (2x + 5)**

```
∴ 7 - 15x + 20 = 3 - 2x - 5  
∴ 27 - 15x = -2 - 2x  
∴ 27 = -2 + 13x  
∴ 29 = 13x  
∴ 29/13 = x  
∴ 2 3/13 = x  

∴ x = 2 3/13  
```

### 10. Fractional linear equations

**x/3 = 4**

```
∴ x = 4 x 3
∴ x = 12
```

**8/x = 4**

```
∴ 8 = 4x
∴ 8/4 = x
∴ 2 = x
∴ x = 2
```

**(5x - 3)/4 = 2**

```
∴ 5x - 3 = 2 x 4
∴ 5x - 3 = 8
∴ 5x = 11
∴ x = 11/5
∴ x = 2 1/5
```

**3 = (15x + 2)/4x**

```
∴ 4x x 3 = 15x + 2
∴ 12x = 15x + 2
∴ 0 = 3x + 2
∴ -2 = 3x
∴ -2/3 = x
∴ x = -2/3
```
